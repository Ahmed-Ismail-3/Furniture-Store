# Furniture-Store

#### ITI Graduation Project - Furniture Store frontend.

An e-commerce website build using the latest technologies with many functions for both users or admins 

## Packages

*	react
*	react-router-dom
*	react-icons
*	react-notifications-component
*	bootstrap
*	axios
*	jwt-decode
*	redux
*	react-redux
*	redux-devtools-extension

## Installation

Use the package manager [NPM](https://www.npmjs.com/) to install Furniture-Store Dependencies.

```bash
npm install
```

## Usage

```python
# start the server
npm start
```
